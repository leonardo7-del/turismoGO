TravelNow Perú 🌎✈️

Un catálogo web interactivo para explorar destinos turísticos en el Perú. Muestra fotos, precios, información relevante por distrito y resalta las promociones y viajes con descuento de forma clara y atractiva.

✨ Características principales

Catálogo de viajes con imágenes y precios.

Listado detallado con información relevante de cada destino.

Filtros por distrito y categoría para encontrar viajes fácilmente.

Resaltado automático de promociones y diferenciación visual para las ofertas.

Diseño responsive: funciona en computadoras, tablets y móviles.

Tematización peruana: colores, estilos y destinos representativos del país.

🚀 Instalación y uso

Clona este repositorio:

git clone https://github.com/tuusuario/travelnow-peru.git


Abre la carpeta del proyecto:

cd travelnow-peru


Abre el archivo index.html en tu navegador favorito.

(No requiere servidor, es un proyecto estático.)

📂 Estructura del proyecto
travelnow-peru/
│── index.html         # Página principal
│── css/
│   └── estilos.css    # Estilos principales
│── js/
│   └── script.js      # Funciones y filtros
│── img/
│   └── ...            # Imágenes de destinos
└── README.md          # Este archivo

📝 Personalización

Cambia las imágenes dentro de la carpeta img/.

Edita index.html para agregar o quitar destinos.

Modifica script.js para ajustar el filtrado o resaltar promociones.
